# YouTube Video Feed

WordPress plugin that:

- Pulls videos from a YouTube channel (no local video storage)
- Displays them in a grid with search and async pagination
- Opens videos in a modal with YouTube embed + view count

## Install

1. Copy `youtube-video-feed` into `wp-content/plugins/`.
2. Activate **YouTube Video Feed** in WordPress.
